TÀI LIỆU SVN VÀ RULE TRONG THƯ MỤC Rules.

.Tuyệt đối không commit các file trong thư mục bin và obj
        .Tuyệt đối không commit file .suo (file này ẩn)
        .Các file và thư mục trên để vào ignore list hết.
        .Khi commit nhớ ghi log/comment cho mọi file (code, doc, UML..)
        .Nội quy commit sẽ update trong file readme.txt
